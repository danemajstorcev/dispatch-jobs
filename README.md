Dispatch Jobs


Project Description:

Dispatch Jobs is a job listing platform focused on the trucking and dispatch industry. It simulates a real-world logistics job board, allowing users to browse and filter job opportunities in a structured and responsive interface.


Tech Stack / Built With:

- Next.js 
- TypeScript
- Tailwind


Features:

- Job listings interface
- Filtering system for jobs
- Industry-specific UI design (logistics / dispatch focus)
- Responsive layout for all devices
- Optimized routing with Next.js


Getting Started / Installation:

- Clone the repository:
git clone https://github.com/danemajstorcev/dispatch-jobs.git

- Navigate to the project folder:
cd dispatch-jobs

- Install dependencies:
npm install

- Run the development server:
npm run dev

- Open in browser:
http://localhost:3000
